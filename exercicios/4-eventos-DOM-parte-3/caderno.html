Exemplos de eventos DOM = onclick, onmouseout, etc...
Eu havia aprendido como modicar um elemento html inserindo o
evento na proria tag.
Ex: < div onmouseover="onOver(this)" onmouseout="onOut(this)" >click
< script >
function onOver(elemento){
elemento.style.backgroundColor = "red"
}
function onOut(elemento){
elemento.style.backgroundColor = "cornflowerblue"
}
</script >
nessa aula foi apresentado como alterar um conteúdo de uma tag
sem precisar inserir um evento na mesma.
maneira 1:
< script >
function carregado(){
var t = document.getElementById("titulo")
t.addEventListener ("click", mudarTitulo)
}
function mudarTexto(){
this.innerHTML = "ppp"
} </script >
< body onload="carregado()">
< h1 id="titulo">Teste</h1 >

obs: nesse caso o onload foi nessário porque a função estava
acima do objetivo referido por ela, no caso o h1
maneira 2:
< script >
function carregado(){
var t = document.getElementById("titulo")
t.addEventListener("click", mudarTexto)
}
function mudarTexto(){
this.innerHTML = "ppp"
}
</script >
</head >
< body onload="carregado()">
< h1 id="titulo">Teste</h1 >
</body >
</html >

ATENÇÃO: t.addEventListener("click", mudarTexto) == t.onclick = mudarTexto

explicando: Quando usamos o evento addEventListener o evento que será usado como argumento no caso o "click", nao receberá o on, e nenhum outro evento como onmouseout e outro tb não receberiam.
