eventos DOM parte 2

1 - quando eu coloco um atributo de evento html, como por exemplo o onclick dentro de uma tag HTML, e atribruo à este atibruto uma função, eu posso colocar o (this), e não precisar identificar o elemento na função com um getElementsByTagName por exemplo. Entretanto eu devo escrever qqr letra ou palavra nos parênteses da função, para quando eu escrever o código JS eu poder identificar o elemento.
EX:

let h1 = document.getElementsByTagName("h1")[0]
h1.innerText = i.value
}

Neste caso o esta função está se referindo ao código html abaixo

<h1> Clique em mim </h1>
<div onclick="hideH1()">Click </div>
<input type="text" onchange="changeh1(this)">
obs: se eu usar aspas duplas para insirir um código JS dentro de um atributo de evento, eu tenho que colocar aspas simples para especificar este código JS.
Ex:
<h1 onclick = "console.log('h1 foi clicado')">
