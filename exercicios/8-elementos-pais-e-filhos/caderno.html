addEventListener e target

h1.addEventListener ("click", qualquerFuncao)

isso serve para adicionar um evento, numa função no javaScript, sem ter que especificar o evento na tag HTML.o addEventListener sempre é precedido por uma variável, a qual será alvo no evento no caso o h1.

O "click", especifica qual é o evento, no caso o onclick. Porém quando usamo eventos no addEventListener sempre o colocamos entre aspas e sem o on.

qualquerFuncao, é pra criar a função que acontecerá o evento click, e o que ocorrerá depois do objeto, no caso o h1, ser clicado

function cor (e){
let t = e.target
t.style.color = "red"
}

"e" e "target"

o (e), que poderia ser qualquer outra letra ou palavra, porém por convensão usamos a letra e, faz referência ao evento DOM, no caso o "click". Por isso, ele fica entre parêntes.

o target, refere-se a variável h1. Pois é a variável que precede o h1.addEventListener. Logo usamos let = e.target para afirma que o elemento(h1), receberá o evento(e). E a partir de aí poder alterar o que for no caso a cor da tag quando ela for clicada.

window.onload = function(){
let h1 = document.getElementsByTagName("h1")[0]
h1.addEventListener("click", cor)
}

function cor (e){
let t = e.target
t.style.color = "red"
}
