JSON.parse

transforma um objeto em string

JSON.stringify

transforma uma string num objeto

Ex:let n = {
nome:"fulano",
idade: 20
}
let b = JSON.stringify(n)
let a = JSON.parse(b)
console.log(a)
console.log(b)
let a2 = '{"nome":"fulado", "idade": 20}' (Esta é forma que raramente será usada para transformar uma string num objeto, atenção as aspas.)
let b2 = JSON.parse(a2)
console.log(b2)
